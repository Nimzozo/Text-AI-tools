

# Text-AI-tools • ![Pollinations badge](https://img.shields.io/badge/Built%20with-Pollinations-8a2be2?style=for-the-badge&logoColor=white&labelColor=6a0dad)

**Text-AI-tools** is a set of text tools that uses AI via the [Pollinations API](https://pollinations.ai).

🌐 **[View Live Demo](https://nimzozo.github.io/Text-AI-tools/)**

## Features

### Tech

- **Lightweight interface** with pure HTML, CSS and JavaScript.
- Uses LLMs via the **Pollinations API**.

### Text tools

- **Explain:** Condense long text to a short summary.
- **Format:** Convert arbitrary input into Markdown.
- **Improve:** Polish clarity, grammar, and tone.
- **Translate:** Convert text into another language.

## Usage

1. Visit the live demo at [Text-AI-tools](https://nimzozo.github.io/Text-AI-tools/).
2. Connect to the [Pollinations API](https://pollinations.ai).
3. Enter your text and select the desired tool.
4. Click the "Run" button to process the text.
5. View the output in the designated area.
