# Text-AI-tools

**Text-AI-tools** is a set of tools that allows to format into well-structured Markdown using AI via the [Pollinations API](https://pollinations.ai).

## Features

- Convert arbitrary input into well-structured Markdown (headings, lists, code blocks, links).
- Review and improve clarity, tone, and structure of input text.
- Summarize input.
- Translate content between languages.
